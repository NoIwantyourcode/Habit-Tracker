# Habit Tracker

Do you have any habits that you may want to leave... Or do you have any habits you may want to gain, Well now with habit tracker you can now track your daily activities and habits! 

Be empowered to leave the habits you want to leave and gain new, better habits that you want!

# Features
here are all the features of the website!

1. Add/delete habits - add your daily tasks/habits to track it
2. Daily check-in - Mark a habit as done evryday
3. Streak Tracking - For every consecutive day you do a habit, This will track it
4. Best Streak - this will show your longest streak of all time for a habit
5. Completion rate - This will show the amount of habits you have done today and how many more you need to do
6. 30 day / 7 day heatmap toggle - shows on which days you did a certain habit and on what days you didn't
7. Edit habit name - Entered the name wrong? Change it by pressing the edit button
8. Habit reordering via drag and drop - Reorder your habits in terms of priority
9. Overall stats summary - shows overall statistics of your habits
10. Progress bar - represents your completion rate
11. Habit categories with colors - categorize your habits by giving them colors
12. Browser notifications/reminders - With this, You will never forget to do a habit!
13. Dark/light mode toggle - Switch the theme based on your environment

# Tech Stack

This website uses 3 different coding languages with different purposes, The main.html file is used to structure the website and create the visual elements. The style.css file is used to make those visual elements in main.html look good. The app.js file makes the entire website work and functions as the backend.

# How to contribute

1. clone this github repo
2. make the changes you want to make (You can implement visual changes in style.css, structure changes in main.html and backend/logic changes in app.js)
3. test the website to make sure you didn't break anything
4. submit a pull request here!
